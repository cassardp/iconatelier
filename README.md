# IconAtelier

An app icon editor built with Swift and SwiftUI.

Compose icons from editable layers — text, shapes, and imported images. Everything stays editable until you export. Optional AI text-to-image generation and a community gallery for sharing icons.

## Features

- Editable layers: text, shapes, and imported images
- Fills: solid colors, linear, radial, and mesh gradients
- Borders, drop shadows, and per-layer transforms
- Optional AI text-to-image generation (bring your own OpenAI key)
- Optional community gallery to share and re-import icons
- Export to Apple App Icon Set, Google Play, web favicons, and single PNG

## Tech

- Swift / SwiftUI — iOS 18+
