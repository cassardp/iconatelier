# IconAtelier

**App Icon Maker**

An editor for designing app icons. Compose icons from editable layers — text, shapes, and imported images — then export for iOS, macOS, Android, and the web.

## Features

- Editable layers: text, shapes, and imported images
- Fills: solid colors, linear, radial, and mesh gradients
- Borders, drop shadows, and per-layer transforms
- Optional AI text-to-image generation
- Optional community gallery
- Export to Apple App Icon Set, Google Play, web favicons, and single PNG

## Available On

- iOS 18+
